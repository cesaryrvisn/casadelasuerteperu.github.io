Lee antes de editar — Casa de la Suerte Perú
========================================
Archivos incluidos:
- index.html   -> página principal (carga automáticamente).
- content.json -> archivo editable con todos los textos y enlaces.
- favicon.svg  -> icono del trébol dorado.

Editar desde GitHub (recomendado):
1. Entra a tu repositorio en GitHub.
2. Haz clic en content.json (aparece en la lista de archivos).
3. Toca el lápiz (Edit) y cambia el texto que quieras.
4. Al final, baja y haz Commit changes. Tu sitio mostrará los cambios en segundos.

Editar desde el celular (mis archivos):
1. Extrae el ZIP y abre content.json con un editor de texto (o app Notas).
2. Modifica los textos (respeta la estructura JSON).
3. Sube el content.json modificado al repo (Add file -> Upload files) o reemplázalo en la web de GitHub.

Nota: El sonido de monedas se activa SOLO al presionar el botón "Compra tu ticket".
